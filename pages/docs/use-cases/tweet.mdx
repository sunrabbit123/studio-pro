Gather and summarize tweets from influencers in specific fields like economics or politics, then compile them into a document or share them.

### Step 1. Authentication

Register or log in on the product page. (https://studio-pro.wrtn.ai)

<br />

### Step 2. Request

In the prompt field, type a request to summarize the tweets of people whose tweets you want to gather all at once. Please provide the required account information at this time.

![](/studio-pro/images/docs/use-cases/tweet/step-2-1.png)

<br />

### Step 3. Review

Check the quantity and content of the information found by the agent and provide additional instructions as needed.

![](/studio-pro/images/docs/use-cases/tweet/step-3-1.png)

![](/studio-pro/images/docs/use-cases/tweet/step-3-2.png)

<br />

### Step 4. Save

Please input how to save or send the information to use it, and provide any additional information needed for this purpose.

![](/studio-pro/images/docs/use-cases/tweet/step-4-1.png)

![](/studio-pro/images/docs/use-cases/tweet/step-4-2.png)

<br />

### Step 5. Workflow

Request to create a workflow for detailed editing and repetitive use of this task.

![](/studio-pro/images/docs/use-cases/tweet/step-5-1.png)

<br />

### Step 6. Execution

Run the workflow to check the results.

![](/studio-pro/images/docs/use-cases/tweet/step-6-1.png)

<br />

### Step 7. Format Selection

Please select the export format you want.

![](/studio-pro/images/docs/use-cases/tweet/step-7-1.png)

<br />

### Step 8. Naming

Enter the name and description of the content to be exported, then click the Next button in the upper-right corner.

![](/studio-pro/images/docs/use-cases/tweet/step-8-1.png)

<br />

### Step 9. Preview

Check the preview of the content to be exported, then click the Next button in the upper-right corner.

![](/studio-pro/images/docs/use-cases/tweet/step-9-1.png)

<br />

### Step 10. Set Up

Please set the conditions for automatically running the workflow repeatedly.

![](/studio-pro/images/docs/use-cases/tweet/step-10-1.png)

<br />

### Step 11. Dashboard

You can check the generated widget when you enter the dashboard.

![](/studio-pro/images/docs/use-cases/tweet/step-11-1.png)
