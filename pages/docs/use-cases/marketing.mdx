Create a workflow that repeatedly generates marketing copy and image materials based on the information you input. Build a widget from this workflow to view the automated results on the dashboard.

### Step 1. Authentication

Register or log in on the product page. (https://studio-pro.wrtn.ai)

<br />

### Step 2. Request

In the prompt field, type a request to generate marketing copy and images for your product.

![](/studio-pro/images/docs/use-cases/marketing/step-2-1.png)

<br />

### Step 3. Info Input

Please provide the detailed information requested by the agent to perform the task.

![](/studio-pro/images/docs/use-cases/marketing/step-3-1.png)

<br />

### Step 4. Canvas Access

When the agent creates the workflow, click the “Go to Workflow” button to navigate to the Workflow Canvas.

![](/studio-pro/images/docs/use-cases/marketing/step-4-1.png)

<br />

### Step 5. Customization

Click on the node and enter the values you want to change or add from the pre-filled template.

![](/studio-pro/images/docs/use-cases/marketing/step-5-1.png)

![](/studio-pro/images/docs/use-cases/marketing/step-5-2.png)

![](/studio-pro/images/docs/use-cases/marketing/step-5-3.png)

<br />

### Step 6. Execution

Click the Run button in the upper-right corner and check the generated results in the Execution History tab.

![](/studio-pro/images/docs/use-cases/marketing/step-6-1.png)

<br />

### Step 7. Update

To reuse the workflow and change the generated content, click the existing node and update the entered values to the desired content.

![](/studio-pro/images/docs/use-cases/marketing/step-7-1.png)

<br />

### Step 8. Export

Check the newly generated results in the Execution History tab, and click Export to register this workflow as a widget.

![](/studio-pro/images/docs/use-cases/marketing/step-8-1.png)

<br />

### Step 9. Format Selection

Please select the export format you want.

![](/studio-pro/images/docs/use-cases/marketing/step-9-1.png)

<br />

### Step 10. Naming

Enter the name and description of the content to be exported, then click the Next button in the upper-right corner.

![](/studio-pro/images/docs/use-cases/marketing/step-10-1.png)

<br />

### Step 11. Preview

Check the preview of the content to be exported, then click the Next button in the upper-right corner.

![](/studio-pro/images/docs/use-cases/marketing/step-11-1.png)

<br />

### Step 12. Set Up

Please set the conditions for automatically running the workflow repeatedly.

![](/studio-pro/images/docs/use-cases/marketing/step-12-1.png)

<br />

### Step 13. Dashboard

You can check the generated widget when you enter the dashboard.

![](/studio-pro/images/docs/use-cases/marketing/step-13-1.png)
