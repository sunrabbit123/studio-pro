Review the GitHub repository, identify areas for improvement or additional features, and suggest ways to contribute to the project.

### Step 1. Authentication

Register or log in on the product page. (https://studio-pro.wrtn.ai)

<br />

### Step 2. Request

In the prompt field, type a request to review a specific GitHub repository and suggest potential ideas for contributing to the project.

### Step 3. Selection

Please select a specific project from the suggested ones and provide guidance on what to consider during the selection.

![](/studio-pro/images/docs/use-cases/github/step-3-1.png)

![](/studio-pro/images/docs/use-cases/github/step-3-2.png)

### Step 4. Initiation

In the prompt field, type a request to initiate a new development phase by branching off, documenting progress incrementally, and submitting the work for peer review. Include guidance on involving specific team members for feedback and mention any preliminary research or tools that may be required.

![](/studio-pro/images/docs/use-cases/github/step-4-1.png)

![](/studio-pro/images/docs/use-cases/github/step-4-2.png)

![](/studio-pro/images/docs/use-cases/github/step-4-3.png)

<br />

### Step 5. Commit

Complete the implementation and commit the final changes to the branch.

![](/studio-pro/images/docs/use-cases/github/step-5-1.png)

<br />

### Step 6. Pull Request

Create a pull request for the completed work and submit it.

![](/studio-pro/images/docs/use-cases/github/step-6-1.png)

<br />

### Step 7. Link

Complete all the steps and provide a link to the pull request.

![](/studio-pro/images/docs/use-cases/github/step-7-1.png)

<br />

### Step 8. Progress

Review the progress of the project through the provided link.
